## Summary
CUDA Release Notes document the current CUDA Toolkit release, component versions, compatibility requirements, new features, resolved issues, known issues, and library changes. The current docs page is for CUDA Toolkit 13.2 Update 1.

## Detail
The release notes are important because CUDA Toolkit components are versioned independently. A CUDA Toolkit release can include specific versions of compiler components, runtime libraries, developer tools, Nsight tools, and CUDA-X libraries.

For this wiki, CUDA release notes should be linked whenever an answer depends on exact toolkit, driver, component, library, or known-issue details. It complements [[CUDA-Compatibility]], which captures driver/toolkit compatibility rules.
