## Summary
NVIDIA API Documentation is the public reference hub for NVIDIA APIs, including NIM and CUDA-X microservices. It connects hosted cloud endpoints, build.nvidia.com prototyping, and product integration guidance for developers who want to call NVIDIA AI services from applications.

## Detail
The docs page positions NVIDIA APIs as a way to integrate NVIDIA capabilities with a few lines of code and prototype against cloud-hosted NVIDIA NIM endpoints. In the wiki, this page acts as the API-facing companion to [[NVIDIA-NIM]], [[NVIDIA-Cloud-Functions]], and [[NVIDIA-NGC-Catalog]].

This matters because NVIDIA content increasingly ships in several forms: hosted API endpoints, downloadable/self-hosted NIMs, NGC catalog artifacts, and cloud platform services. API documentation is the entry point for application developers who want to test an endpoint before deciding whether to self-host.
