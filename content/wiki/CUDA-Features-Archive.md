## Summary
The CUDA Features Archive is NVIDIA's current reference page for looking up CUDA feature availability across toolkit and driver releases. It is not a recommendation to use older toolkit releases; it is a compatibility and planning reference for developers maintaining applications across CUDA versions.

## Detail

### Purpose
CUDA feature support changes across toolkit, driver, and GPU architecture releases. The features archive helps developers and platform owners answer when a feature appeared, what version family introduced it, and how it relates to deployment planning.

### NVIDIA context
Use this page alongside [[CUDA-Release-Notes]] and [[CUDA-Compatibility]]. The release notes tell what changed in a current CUDA release; CUDA Compatibility explains driver/toolkit compatibility; the features archive gives historical feature availability context without making old releases canonical.
