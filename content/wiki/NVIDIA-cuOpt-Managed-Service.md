## Summary
NVIDIA cuOpt Managed Service is NVIDIA's hosted/service-oriented documentation surface for cuOpt routing and logistics optimization. It covers quick-start setup, credentials, Python thin client usage, CLI/API usage, data requirements, routing constraints, troubleshooting, and best practices.

## Detail
This page is separate from [[cuOpt]] because the existing cuOpt page covers the GPU-accelerated optimization library and solver concept, while the managed service page covers cloud/API access patterns and service-specific workflows.

The docs emphasize vehicle routing and optimization data requirements such as cost matrices, waypoint graphs, fleet features, task features, and solver configuration. It belongs near [[NVIDIA-API-Documentation]] because API access and credentials are part of the managed-service flow.
